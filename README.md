
# IMEDIA24 FRONTEND TEST

Built with React/ Vite / TypeScript/ Redux / Saga / Tailwind / Jest /Cypress


## Instalation
 
```bash
 cd imedia24_frontend_test
 npm install  
 
```
or
```bash
 cd imedia24_frontend_test
 yarn install  
 
```

  npm run dev
